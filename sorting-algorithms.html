Sorting Algorithms Algoritmos de Ordenamiento

In this game you will be able to choose a sorting algorithm and use it with a set of numbers that can be random or not, then, you will be able to see how it works step by step. En este juego podrás elegir un algoritmo de ordenamiento y usarlo con un conjunto de numeros que puede ser aleatorio o no, después, podras ver como se comporta paso a paso.

Controls Controles

- Mouse
- Keyboard
 - Ratón
- Teclado
